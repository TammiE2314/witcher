泡泡信 · Bubble Letters（最终版）

✅ 你要的导出编辑器：每个字段一行「左框 + 示意 + 右框」，顶部/底部也可填
✅ 支持上下箭头调整导出字段顺序（不影响模板字段顺序）
✅ 手机/电脑可用：默认本机存储；需要两端同数据可用 Supabase 同步（可选）

打开方式（建议）
- 电脑：双击 index.html
- 手机：建议上传到 GitHub Pages / Netlify / Vercel 或者用“本地服务器”类 App 打开（因为部分浏览器对 file:// 的 localStorage 有限制）

导出说明
- 顶部行：例如  xxxx：{
- 每行：左框 + (泡泡真实文本) + 右框
- 底部行：例如 },
